*Instalar git e iniciar sesion en GitHub mediante VSCode*
Instala Git

# Identifícate en la terminal de VSCode
```c
git config --global user.name "TuNombreDeUsuario"
git config --global user.email "tucorreo@ejemplo.com"
```

# Vincular repositorio
```c
git remote add origin https://github.com/usuario/nombre_repositorio.git
git branch -M main
git push -u origin main
```

# Guardar versiones del código desde VSCode y compartirlo a tu repositorio de GitHub
```c
git init
git add archivo.py
git commit -m "Mensaje"
git push origin main
```
