# Cómo vincular tu código de VSCode a tu repositorio de GitHub

### 1 — Instala Git en tu computadora

### 2 — Identifícate en la terminal de VSCode
```c
git config --global user.name "TuNombreDeUsuario"
git config --global user.email "tucorreo@ejemplo.com"
```

### 3 — Vincular repositorio
```c
git remote add origin https://github.com/tu_nombre_de_usuario/nombre_repositorio.git
git branch -M main
git push -u origin main
```

### 4 — Guardar versiones del código desde VSCode y compartirlo a tu repositorio de GitHub
```c
git init
git add archivo.py
git commit -m "Mensaje"
git push origin main
```
